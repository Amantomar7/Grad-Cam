# GradCAM

GradCAM is a lightweight Python library that provides **model-agnostic Grad-CAM visualizations** for vision models built with **PyTorch** and **Hugging Face Transformers**.

It is designed to work out-of-the-box with **custom user-defined models** as well as popular pretrained vision architectures, without requiring architecture-specific code or boilerplate.
